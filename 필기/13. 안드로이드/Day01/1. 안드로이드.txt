

@ 안드로이드
-> 안드로이드는 구글(Google)에서 만든 스마트폰용 운영체제(OS)
-> 휴대용 단말기를 위해 만들어진 것일 뿐만아니라 다양한 애플리케이션을 만들어 설치하면 실행 될 수 있도록
구성된 애플리케이션 플랫폼(Platform)
-> 자바 개발 언어를 사용하기 때문에 자바 개발자는 손 쉽게 안드로이드 앱 개발을 할 수 있음

@ 안드로이드 개발 방식
-> 네이티브 앱
- 해당 플랫폼에서 요구하는 언어 및 API를 사용하여 만드는 앱
- 성능이 웹앱, 하이브리드 앱에 비하여 가장 좋음


-> 모바일(웹) 앱
- 기본적인 웹 사이트를 모바일에서 볼 수 있는 형태로 바꾼 웹
- 모바일 브라우저를 통해 접속한 웹 사이트라고 볼 수 있음
- 모바일 웹(웹 앱)은 마켓에 등록할 수 없음

-> 하이브리드 앱
- 네이티브앱 + 웹앱
- 네이티브 API와 브라우저 API를 이용한 다양한 개발이 가능함
- 웹 개발 기술을 사용하여 앱을 개발할 수 있음
- 웹에서 사용하는 UI 개발 기술을 응용하여 개발할 수 있음
- 자바 웹 개발자는 하이브리드 앱을 쉽게 개발할 수 있는 장점이 있음
