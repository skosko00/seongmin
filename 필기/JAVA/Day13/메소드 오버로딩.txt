

@ 메소드 오버로딩
-> 자바의 메소드 호출에는 메소드명과 매개변수 개수, 타입을 보고 호출할 메소드를 찾음
-> 그렇기 때문에 같은 이름의 메소드명이라고 하더라도 매개변수의 개수가 다르거나
타입이 다르다면 동일한 메소드명으로 제작이 가능함

	Test1 t = new Test1();	
		t.testMethod();		// 안녕하세요 테스트 메소드 입니다.
		t.testMethod(10);	// 호출 값은 10입니다.
		t.testMethod(10,20);	// 2개의 값은 10과 20이며 합한 값은 30입니다.
