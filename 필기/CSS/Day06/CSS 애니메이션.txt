

@ 애니메이션
-> 애니메이션은 속성을 점차적으로 변화시켜 움직이는 효과를 내는 기법


## 애니메이션에서 사용하는 속성 ##

animation-delay
-> 엘리먼트가 로드 되고 나서 언제 애니메이션이 시작될지 지정

animation-direction
-> 애니메이션이 종료되고 다시 처음부터 시작할지 역방향으로 진행할지 지정

animation-duration
-> 한 싸이클의 애니메이션이 얼마나 걸려 일어날지 지정

animation-iteration-count
-> 애니메이션이 몇 번 반복될지 지정
-> infinite로 지정하면 무한히 반복할 수 있음

animation-name
-> 애니메이션의 중간 상태를 지정
-> 중간 상태는 @keyframes 규칙을 이용하여 기술

animation-timing-function
-> 중간 상태들의 전환을 어떤 시간간격으로 진행할지 지정



## 전환 ##
-> CSS의 hover를 이용하여 글자의 크기 전환을 하는 것


## transition-duration ##
-> 변환시킬 지속 시간을 지정할 수 있음


## transition-delay ##
-> delay 속성을 주게 되면 애니메이션이 시작될때까지의 시간을 설정 할 수 있음


## 역전환 ##
-> 커서가 올라갈때 글씨가 커지는 전환 기법을 적용하게 되면 글씨가 커지게 됨
-> 이때, 글씨가 커진후 커서가 빠져나가게 되면 글씨가 빠른 속도로 줄어듦
-> 역전환 기법을 이용하여 글씨의 크기가 원래대로 복귀할때 속도를 줄 수 있음
	[부드러운 효과를 줄 때 사용함]


@ 키 프레임 사용
-> 키 프레임을 사용하면 애니메이션의 중간 설정을 줄 수 있음

ex)
    div:active{
    animation-name: hello;
    animation-duration: 2s;
    }
    
    @keyframes hello
    {
        from{
            font-size: 10px;
        }
        50%{
            background-color: red;
        }
        to{
            font-size: 50px;
        }
    }

## 애니메이션 앞뒤로 움직이기 ##
-> animation-deirection 속성을 alternate로 지정하면 앞뒤로 반복 됨


@ transform
-> transform을 이용하면 회전, 크기 조절, 이동, 기울기 등의 여러 작업을 할 수 있음
-> 다양한 속성을 가지고 2D나 3D를 표현할 수 있음


## rotate ##
-> 지정한 각도만큼 요소를 시계방향(양수값)이나 반대방향(음수값)으로 회전 시키는 함수

선택자 {
	transform : rotate(값deg);
}
