

@ 2차원 배열
-> 2차원 배열은 인덱스를 2개 사용하여 행과 열을 표현하는 것

ex) int [][] arr = new int[3][4];

int [][]arr = new int[행][열]
	※ 가로가 행, 세로가 열


@ 2차원 배열의 초기화

int [][] arr = {1,2,3,4,5,6,7,8}; //2x4 4x2인지 모르기 때문에 불가능 함

int [][] arr = {{1,2,3,4},{5,6,7,8}};



@ 2차원 배열의 공간 생성

int [][] arr = new int[3][4];	// 3개의 행과 4개의 열이 만들어 짐
