

@ 표 관련 태그
-> 표를 만들때 사용되는 태그


## table 태그 ##
-> 웹 문서에서 자료를 정리할때 자주 사용 (표 형태)
-> 행과 열로 이루어져 있고, 행과 열이 만나는 지점을 셀이라고 함

<tr> 태그와 <th>,<td> 태그를 이용함


<tr> : 한 개의 행(row)를 만드는 태그
<th> : 열에 대한 제목을 표시하는 태그, 중앙 정렬 및 굵게 표시 됨
<td> : 한 개의 열을 만드는 태그


## table 병합 속성 ##

rowspan		: <td> 태그의 속성으로 지정한 행만큼 행을 병합
colspan		: <td> 태그의 속성으로 지정한 열 만큼 열을 병합

사용법

<td rowspan="숫자"></td>
<td colspan="숫자"></td>



## 그외 테이블 관련 태그 ##

테이블을 각 각 나누어서 활용 할 수 있음

<thead>
-> 테이블 구조를 나눌때 머리 부분 태그
-> 한개만 존재 할 수 있음

<tbody>
-> 테이블 구조를 나눌때 몸체가 되는 부분
-> 여러개가 존재할 수 있음

<tfoot>
-> 테이블 구조를 나눌때 마지막이 되는 부분
-> 한개만 존재 할 수 있음
